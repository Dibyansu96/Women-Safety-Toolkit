# Women-Safety-Toolkit

A modern, feature-rich **Women Safety Toolkit** built to provide quick access to emergency assistance during critical situations. Designed as a lightweight and responsive Progressive Web App (PWA), this application combines essential safety tools such as emergency alerts, live location sharing, evidence capture, and offline accessibility into a single platform.

Whether online or offline, the toolkit is designed to help users respond quickly and efficiently during emergencies.

---

## ✨ Key Features

### 🚨 Emergency Response

* **SOS Alarm** with loud siren, flashlight, and device vibration
* **Panic Mode** that instantly activates emergency actions
* **Quick Dial** for emergency services and trusted contacts
* **SMS & WhatsApp Prefill** for rapid emergency communication

### 📍 Location & Communication

* Real-time **Live Location Sharing**
* One-tap location sharing through WhatsApp or SMS
* Emergency Contacts Directory with customizable contacts

### 🎥 Evidence Collection

* **Camera Evidence Capture**
* **Audio Recording** for incident documentation
* Easy sharing of recorded evidence via supported messaging platforms

### 📞 Personal Safety Tools

* **Fake Call Simulator** for escaping uncomfortable situations
* Safety Tips and First Aid Guides
* Text-to-Speech support for improved accessibility

### 🎨 User Experience

* Responsive design for mobile and desktop devices
* Dark Mode, Light Mode, and High Contrast accessibility theme
* Offline functionality through Progressive Web App (PWA) support
* User Authentication interface
* Local storage for contacts and application settings

## 📱 Progressive Web App (PWA)

The application can be installed directly on supported devices and works even with limited or no internet connectivity.

**PWA Features**

* Offline access
* Installable on mobile and desktop
* Fast loading with service worker caching
* Native app-like experience

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/women-safety-toolkit.git
```

### Navigate to the Project

```bash
cd women-safety-toolkit
```

### Launch the Application

Simply open `index.html` in your browser, or use a local development server such as:

```bash
npx serve
```

or

```bash
python -m http.server
```
